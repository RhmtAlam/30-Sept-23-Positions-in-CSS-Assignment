What Is Absolute Positioning?

You may position your element exactly where you want it with absolute positioning. The initial parent element that is positioned relatively (or absolutely) serves as the reference point for absolute positioning. The element with absolute position will be positioned in relation to the HTML element (the page itself) if there is no positioned parent element. Making ensuring absolute positioning is not misused is a crucial consideration because doing so can result in a maintenance nightmare. What Is Relative Positioning?

Without adding any additional positioning properties (top, bottom, right, or left), nothing will happen when you set the position relative to an element. When you add a position other than relative, like left: 20px, the element will shift 20px to the right of where it normally is. You can see that this element is related to itself in this instance. No other element on the layout will be impacted by the element's movement. Relative to an element restricts the range of absolutely positioned child components, thus keep that in mind while setting position.
